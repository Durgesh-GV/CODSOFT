# Task 4–Movie Recommendation System

## Description

This project implements a simple **movie recommendation system**
using Python. It recommends movies based on **genre similarity**
by applying **cosine similarity** on text data.

## Features

* Content-based movie recommendation
* Genre-based similarity matching
* Case-insensitive movie search
* Console-based user interaction
* Beginner-friendly implementation

## Technologies Used

* Python
* Pandas
* Scikit-learn

## Dataset Used

This project uses a **movie dataset obtained from Kaggle**.

**Dataset Source:**
Kaggle – Movie Recommendation Dataset

**Note:**
The dataset is **not uploaded** to this repository.
Please download the dataset from Kaggle and place
`movies.csv` in the **same folder** as `recommentation system.py`
before running the program.
