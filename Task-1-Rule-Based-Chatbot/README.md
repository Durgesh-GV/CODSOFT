## Description
This project implements a simple rule-based chatbot using Python.
It responds to user inputs such as greetings, date, time, and exit
commands using predefined rules and if-else statements.

## Features
- Greets the user by name
- Responds to common greetings
- Displays current date and time (IST)
- Handles exit commands politely
- Beginner-friendly implementation

## Technologies Used
- Python
- datetime module

## How to Run
1. Open Command Prompt or Terminal
2. Navigate to this folder
3. Run the command:
   python chatbot.py
